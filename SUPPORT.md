# Support

This project is open source, but it does not come with guaranteed support or
service-level commitments.

## Questions and Usage Help

Use GitHub issues for:

- reproducible bugs;
- documentation problems;
- clear feature requests;
- setup problems that appear to affect the project rather than one local
  environment.

Before opening an issue, check the README, existing issues, and current
documentation.

## Security Reports

Do not report vulnerabilities through public issues. Follow `SECURITY.md`.

## Production Operations

The deployment and infrastructure files in this repository are provided for the
official Fiestaaa deployment and as implementation reference. Maintainers do
not provide operational support for unrelated self-hosted deployments.

If you run a fork, you are responsible for:

- secrets and key rotation;
- database backups and restores;
- monitoring and alerting;
- legal, privacy, and compliance obligations;
- third-party service accounts and billing.

## Maintainer Availability

Issues and pull requests are reviewed on a best-effort basis. Clear
reproduction steps, focused patches, and passing tests make review faster.
